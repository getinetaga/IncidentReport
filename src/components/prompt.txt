Create a mobile application that enables users to report incidents in real time by automatically capturing the precise location of the event using GPS. The app should allow users to categorize and rank situations by urgency, including emergencies, crime reports, accidents, and general community issues. Each report should support additional details such as descriptions, photos, videos, and timestamps to help authorities or responders assess the severity of the situation quickly. The system should prioritize high-risk incidents for faster response, provide a simple and intuitive user experience, and ensure strong data privacy, security, and reliability during critical situations.
